# Waffle Bar Chain Ordering Website

## Project Description  
A website for ordering dishes from the Waffle Bar chain, featuring a polished homepage, order page, detailed menu, and contact page. The site is beautifully designed, responsive, and includes smooth animations for an enhanced user experience.

## Technologies Used  
- **Frontend**: HTML, CSS  
- Responsive design with animations  
- User-friendly and visually appealing interface

## Installation and Running Instructions  
To run the website locally, simply download the project files and open the HTML file from the project folder in your preferred browser. No additional installation is required.

## Site Structure  
- Home Page  
- Order Page  
- Menu Page  
- Contact Page
